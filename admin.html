Database Setup (Supabase)
⚡ Running locally
Right now your data is saved in the browser (localStorage). To make it permanent and accessible from anywhere, connect Supabase — it takes about 5 minutes and is completely free.
1
Create a free Supabase account
Go to supabase.com → New project → Name it "pawdia" → Choose a strong DB password
2
Create your tables
In Supabase → SQL Editor → paste this and run:
-- Products table create table products ( id bigserial primary key, name text, brand text, category text, price int, mrp int, ico text, description text, approved boolean default true, created_at timestamptz default now() ); -- Places table create table places ( id bigserial primary key, name text, type text, address text, score numeric, open boolean, summary text, tags text[], created_at timestamptz default now() ); -- Blog posts create table blog_posts ( id bigserial primary key, title text, category text, content text, status text default 'draft', created_at timestamptz default now() ); -- Lost dogs create table lost_dogs ( id bigserial primary key, dog_name text, breed text, last_seen text, owner_phone text, status text default 'missing', created_at timestamptz default now() );
3
Copy your keys into Settings
Supabase dashboard → Settings → API → copy Project URL and anon public key → paste in the Settings panel above
4
That's it — data now saves permanently
Any product, place or blog post you add here will instantly appear on pawdia.in and the app. No code needed ever again.
